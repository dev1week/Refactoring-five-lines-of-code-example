## 정리

**규칙 1.** 모든 함수는 5줄 이하로 작성한다.

**규칙 2.** 하나의 함수 안에서는 호출과 전달 둘 중에 하나만 해야한다.

**규칙 3.** if문은 함수 시작에만 배치하자. 단, `if`, `else if`,`else`구조를 분할하면 안된다.
